# Task Management System

A simple command-line application that allows users to create, manage, and track tasks efficiently using Python.

---

## Features

- Create tasks with title, description, priority, due date, and tags
- Update task status, priority, and due date
- Filter tasks by status, priority, or overdue
- Manage tags for tasks
- View task details and statistics
- Persistent storage using JSON

---

## Technologies Used

- Python 3.11
- Standard Library (argparse, json, datetime)
