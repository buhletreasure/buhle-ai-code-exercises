# FAQ

## What is this system?
A command-line task management tool used to create and manage tasks.

## How do I create a task?
python cli.py create "Task"

## How do I view tasks?
python cli.py list

## How do I complete a task?
python cli.py update-status <task_id> done

## Where is data stored?
In tasks.json

## Why is my command not working?
- Check command spelling
- Check task ID
- Ensure Python is installed
